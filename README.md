#  DVC basic demo

## create the fresh annconda env 

```bash 
        $ conda create -n dvc python=3.6 -y 
        $ conda activate dvc 
```

## command in use commanly 

```bash 
        $ pip install dvc 
        $ git init
        $ dvc init  
```
- after initializing the 'DVC' , there will be two files created namely ".dvc" , ".dvcignore"
